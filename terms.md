# CrashyBot — Privacy Policy

**Effective Date:** 2025-11-28

## 1. Introduction  
This Privacy Policy explains how CrashyBot (“the Bot”) collects, uses, stores, and protects user data when operating on the Discord platform. By using the Bot, you agree to the collection and use of information in accordance with this policy.

## 2. Data We Collect  
CrashyBot may collect **only the minimum technical data required for its functionality**, which may include:

- Discord User ID  
- Discord Server (Guild) ID  
- Discord Channel ID  
- Command usage data (which command was used and when)

CrashyBot **does NOT collect**:
- Real names  
- Email addresses  
- IP addresses  
- Passwords  
- Payment or financial information  
- Private message content  
- Any sensitive personal data  

## 3. How We Use Data  
Collected data is used strictly for:
- Ensuring correct command functionality  
- Debugging and fixing technical issues  
- Preventing abuse or spam  
- Improving future features  

Data is **never used for advertising, profiling, or tracking outside Discord**.

## 4. Data Storage & Security  
- Stored data (if any) is kept only as long as necessary to operate the Bot correctly.  
- Reasonable technical measures are used to protect the data from unauthorized access.  
- The developer does not guarantee absolute security, but best practices are followed.

## 5. Data Sharing  
CrashyBot **does not sell, rent, or share any collected data with third parties** under any circumstances.  
Data is only accessible to the bot developer for maintenance, debugging, and support purposes.

## 6. User Rights  
You have the right to:
- Request a copy of any data stored about you (if applicable).  
- Request deletion of your data at any time.  

Requests must be made through the contact method listed on the bot’s Top.gg page or through the official support contact.

## 7. Children’s Privacy  
CrashyBot is not intended to collect data from users under the minimum age required by Discord’s Terms of Service. By using the Bot, you confirm that you meet Discord’s age requirements.

## 8. Changes to This Policy  
This Privacy Policy may be updated at any time. When updates occur, the “Effective Date” will be revised. Continued use of the Bot after any update means you accept the revised policy.

## 9. Contact  
For privacy-related questions, data requests, or concerns, please contact the developer through the communication method listed on the CrashyBot Top.gg page.
